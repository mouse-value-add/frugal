The rack-rate gap

The cheapest tools for AI.

Tool prices haven't fallen the way model prices have. A Firecrawl scrape costs as much as the gpt-4o-mini call that consumes it. Tavily at $0.008/call is 8× Serper at $0.001/call. SearXNG, running on your own machine, is free. Frugal walks the columns left to right — free / local first, cheap paid as fallback, premium paid only when you opt in.

Every tool call prints the decision in the agent's tool trace: capability, provider chosen, cost paid.

The product

One MCP server. That's it.

Agent stacks talk to Frugal via MCP — Claude Desktop, Cursor, Claude Code, any custom MCP host. Two commands; no other surface.

Get started

Install

One Go binary, ~10 MB. No SaaS. No account. BYOK — Frugal reads OPENAI_API_KEY, ANTHROPIC_API_KEY, GOOGLE_API_KEY, SERPER_API_KEY from your environment and forwards calls upstream. Releases are cosign-signed; the installer verifies the checksum before moving the binary into place. 

$ curl -fsSL https://frugal.sh/install | bash

From source

$ git clone https://github.com/brainsparker/frugal.git && cd frugal && make build

Add Frugal to your agent

$ frugal mcp install
# auto-detects Claude Desktop, Cursor, Claude Code.
# merges 'frugal' into each detected config. idempotent.
